cthulhu
=======

MAPAS DE ENTORNOS MEDIANTE NAVEGACIÓN DIFUSA Y UN SISTEMA DE TELEOPERACIÓN DE UNA PLATAFORMA PIONNER P3-DX

Realizar un mapeo basico y un control difuso de velocidad para la plataforma Pionner P3-DX

Requierimientos:

-Aria y ArNetworking

-Python 2.5 - 2.7

-pyGTK

-openCV

-pyfuzzy
